# Proton Home

I didn't like `account.proton.me/apps`. So I made my own.

## Installation

Set your browser's homepage to `https://josselinonduty.github.io/proton-home/` and voila.

## Features

None, just a simple static page.

> [!NOTE]
> I may add some themes in the future.
